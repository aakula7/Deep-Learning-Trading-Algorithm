# Deep-Learning-Trading-Algorithm

## Introduction
This is a long-term side project of building web application that can be used by retail stock traders who are slowly taking a pie of total number of traders today through the help of accessible applications such as Robinhood and Acorns. We want to build a web application, which helps common people have a better understanding of stocks and the risk they are taking, in hopes that they trade with greater knowledge of the stock market.
